📌 A. Develop Core Microservices

You'll develop two core microservices using .NET Core 8, each as independent APIs:

1. Employee Service (API)
Endpoints:

  • GET /employees Retrieve all employees
  • GET /employees/{id} Retrieve specific employee details
  • POST /employees Create new employee
  • PUT /employees/{id} Update employee details
  • DELETE /employees/{id} Delete employee (soft delete or deactivate recommended)

2. Leave Service (API)
Endpoints:

  • GET /leaves Get leave records
  • POST /leaves Apply for leave
  • PUT /leaves/{id} Approve or Reject leave
  • DELETE /leaves/{id} Cancel leave request

Example Code Structure (.NET 8):

bash

dockerfile

CopyEdit

FROM mcr.microsoft.com/dotnet/aspnet:8.0 AS base

WORKDIR /app

EXPOSE 80

FROM mcr.microsoft.com/dotnet/sdk:8.0 AS build

WORKDIR /src

COPY . .

RUN dotnet restore "EmployeeService.csproj"

RUN dotnet publish "EmployeeService.csproj" -c Release -o /app/publish

FROM base AS final

WORKDIR /app

COPY --from=build /app/publish .

ENTRYPOINT ["dotnet", "EmployeeService.dll"]
